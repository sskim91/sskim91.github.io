mysql 조건부 데이터 입력, 변경

2018-01-10
database, mysql

DB는 기본 키가 중복된 데이터를 입력하는 것을 허용하지 않는다.

예를들어 100건을 입력하고자 하는데 첫 번째 한번의 오류 때문에 나머지 99건도 입력되지 않는 것도 문제가 될 수 있다. Mysql은 오류가 발생해도 계속 진행하는 방법을 제공한다.
