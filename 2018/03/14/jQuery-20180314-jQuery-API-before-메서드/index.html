jQuery API 살펴보기 before 메서드

2018-03-14
before, jQuery

조건에 일치되는 target 요소 앞에 추가될 내용(요소)을 삽입한다.

before() 메서드와 insertBefore() 의 차이

before() - $(target).before('내용 contentToBeInserted');

insertBefore() - $('내용 contentToBeInserted').insertBefore(target);
