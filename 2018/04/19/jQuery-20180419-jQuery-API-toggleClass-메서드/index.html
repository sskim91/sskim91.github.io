jQuery API toggleClass 메서드

2018-04-19
jQuery, toggleClass

toggleClass() 메서드

각 요소가 일치하는 집합에서 class 속성을 추가하거나 제거하는 기능을 한다.

.toggleClass( className )

  • className

    Type: String

    일치하는 각 요소에 대해 토글 될 하나 이상의 클래스 이름 (공백으로 구분됨)

.toggleClass( className, state )

  • className

    Type: String

    One or more class names (separated by spaces) to be toggled for each element in the matched set.

  • state

    Type: Boolean

    클래스를 추가 또는 제거해야하는지 여부를 결정하는 부울 (true / false 가 아닌) 값.

.toggleClass( function [, state ] )

  • function

    Type: Function( Integer index, String className, Boolean state ) => String

    A function that returns class names to be toggled in the class attribute of each element in the matched set. Receives the index position of the element in the set, the old class value, and the state as arguments.

toggleClass() 메서드는 하나 이상의 클래스 이름을 매개변수로 사용합니다. 첫번째 버전에서는 일치하는 요소 안에 class 속성이 이미 존재할 때는 그 class 명을 지웠다. 반대로 인자로 주어진 클래스 속성이 없으면 추가된다.

toggleClass 적용시 $( “div.tumble” ).toggleClass( “bounce” )

그 다음으로 아래 스크립트가 한번 더 실행되면 div 태그의 클래스는 원래대로 thumble만 남게된다.

$( “div.tumble” ).toggleClass( “bounce” )
