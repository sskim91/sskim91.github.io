Bootbox

Bootbox.js 소개Bootbox.js는 필요한 DOM 요소 또는 JS 이벤트 핸들러를 생성, 관리 또는 제거하는 것에 대해 걱정할 필요없이 부트 스트랩 모달을 사용하여 프로그래밍 방식 대화 상자를 만들 수있는 작은 JavaScript 라이브러리이다. 공식사이트Bootbox.js 사용방법 CDN https://cdnjs.cloudflare.com/ajax/libs/bootbox.js/4.4.0/bootbox.min.js 다운로드 위의 공식사이트에가서 다운로드 받아서 본인의 소스에 추가해서 사용하면 된다. Bootbox.js는 bootstrap의 모달을 이용하는 것이므로 bootstrap.js가 필요하고 bootstrap의 모달기능을 사용하기위해서는 jquery가 필요하므로 Bootbox.js를 사용하기 위해서는 bootstrap과 jquery를 추가해주어야 한다. 보통 이런 상황을 Bootbox.js가 bootstrap과 jquery에 의존한다고 한다.
